# Z Lineshape, N_nu, and alpha_s -- ALEPH at sqrt(s) ~ 91.2 GeV

Type: measurement

````text
Measure the Z lineshape parameters (M_Z, Gamma_Z, sigma^0_had), the
number of light neutrino species N_nu from the invisible width, and
extract alpha_s(M_Z) from the hadronic-to-leptonic width ratio R_l,
using archived ALEPH data at sqrt(s) ~ 91.2 GeV.

Classify events as hadronic, e+e-, mu+mu-, or tau+tau-. R_l is a
counting ratio (luminosity cancels). N_nu comes from the invisible
width. alpha_s comes from R_l via the N4LO QCD correction. If scan
data at multiple sqrt(s) points is available, fit the lineshape.

References:
- ALEPH, Eur. Phys. J. C14 (2000) 1
- LEP EWWG, Phys. Rep. 427 (2006) 257
- Baikov et al., Phys. Rev. Lett. 108 (2012) 222003
- Query the LEP corpus for ALEPH Z lineshape and invisible width
````
