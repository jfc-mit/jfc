# R_b, R_c, and A_FB^b -- ALEPH at sqrt(s) = 91.2 GeV

Type: measurement

````text
Measure the partial width ratios R_b and R_c, and the b-quark
forward-backward asymmetry A_FB^b, in hadronic Z decays using
archived ALEPH data at sqrt(s) = 91.2 GeV.

Tag b and c events using lifetime methods (signed impact parameter
significance). Use a double-tag method for R_b. For A_FB^b, determine
the b-quark charge using jet charge in tagged hemispheres and measure
the cos(theta_thrust) asymmetry to extract sin^2(theta_eff). Use
charged particles only (pwflag == 0, highPurity == 1).

References:
- ALEPH, Phys. Lett. B401 (1997) 150
- LEP/SLD EWWG, Phys. Rep. 427 (2006) 257
- Query the LEP corpus for ALEPH R_b, jet charge, and A_FB^b
````
