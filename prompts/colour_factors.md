# QCD Colour Factors from 4-Jet Events -- ALEPH at sqrt(s) = 91.2 GeV

Type: measurement

````text
Measure the QCD colour factor ratio C_A/C_F from angular distributions
in 4-jet events in hadronic Z decays using archived ALEPH data at
sqrt(s) = 91.2 GeV.

Select 4-jet events with the Durham algorithm. Measure angular
distributions sensitive to the colour factors (Bengtsson-Zerwas,
modified Nachtmann-Reiter, and/or alpha_34). Fit to LO QCD matrix
element predictions parametrized by C_A/C_F and T_R/C_F. SU(3)
predicts C_A/C_F = 9/4 and T_R/C_F = 3/8. Use all particles
(charged + neutral) for jet clustering.

References:
- OPAL, Eur. Phys. J. C20 (2001) 601
- DELPHI, Eur. Phys. J. C37 (2004) 1
- ALEPH, Eur. Phys. J. C35 (2004) 457
- Query the LEP corpus for colour factor and 4-jet angular distributions
````
