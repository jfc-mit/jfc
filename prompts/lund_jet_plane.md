# Primary Lund Jet Plane Density -- ALEPH at sqrt(s) = 91.2 GeV

Type: measurement

````text
Measure the primary Lund jet plane density in hadronic Z decays using
archived ALEPH data at sqrt(s) = 91.2 GeV.

Decluster each thrust hemisphere with the Cambridge/Aachen algorithm
and map primary splittings to Lund coordinates (ln 1/Delta_theta,
ln k_t/GeV). Correct to charged-particle level. Use charged particles
only (pwflag == 0, highPurity == 1).

This is the first Lund plane measurement in e+e- collisions.

References:
- ATLAS, Phys. Rev. Lett. 124 (2020) 222002
- ALICE, JHEP 05 (2024) 116
- Query the LEP corpus for jet substructure at LEP

Data: /n/holystore01/LABS/iaifi_lab/Lab/sambt/LEP/ALEPH/
MC:   /n/holystore01/LABS/iaifi_lab/Lab/sambt/LEP/ALEPHMC/
````
